Fonts Used

Inter Regular:
Standard font used across the site, you will see this font used most commonly on body text as well as for the footer links.
Inter Bold:
Bold font used on the header text, you will see this font used most commonly on text in a strong tag. As well as on the header text.
Inter Italic:
Italic font used on the footer text, you will see this font used most commonly on text in a em tag. As well as on the footer text and links featured in the main tag.
Inter Bold Italic:
Bold and Italic font used on text that really needs to be emphasized, you will see this font used most commonly on text in a strong tag that is wrapped in an em tag. As well as on text.
